Our team developed this site as an "API mashup" school project in October of 2021. The goal was to create a service which takes a date as an input, displays a "this-day-in-time" historical event list, and provides a map with the modern-day locations of the events. The page loads with the current date, but also gives the user the ability to input any day of the year. The mashup is achieved by combining various Wikipedia and Google APIs and the source can be found at the GitHub link in the footer.
