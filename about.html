About This Project

Our team developed this site as an "API mashup" school project in October of 2021. The goal was to create a service which takes a date as an input, displays a "this-day-in-time" historical list, and provides a map with the modern-day locations of the events. The page loads with the current date, but also gives the user the ability to input any day of the year.

The two main APIs are Wikipedia and Google Maps.
